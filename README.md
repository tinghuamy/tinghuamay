<!DOCTYPE html>
<html>

<head>

</head>

<body>
  <h1>美发技术网站</h1>
  <p>欢迎来到我们的美发技术网站！</p>
  <img src="hair_style_image.jpg" alt="美发图片示例">
  <p>这里将提供各种美发技术的教程和信息。</p>
  <button onclick="showMoreTechniques()">查看更多技术</button>

  <script src="script.js"></script>
</body>

</html>
